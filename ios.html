Istruzioni per iOS

Fortunatamente iOS supporta nativamente i file iCal, quindi la procedura è molto semplice:

  1. Copiare il link del calendario come descritto nella pagina principale;
  2. Apri Impostazioni => Calendario => Aggiungi account => Altro => Aggiungi calendario;
  3. Incolla l'indirizzo copiato precedentemente e conferma;
  4. Apri il calendario: il gioco è fatto!
